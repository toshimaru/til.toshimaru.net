## 複数のDocker Composeのネットワークをつなぐ

下記の記事が参考になった。

[Dockerネットワークとコンテナ名で実現する複数Docker Compose間の接続方法 \| Enjoy IT Life](https://nishinatoshiharu.com/access-container-via-network/)

`external: true` のオプションを付与

### Docker Compose A

```diff
services:
  web:
    ...
+   networks:
+     - external.group
+     - default
  db:
    image: mysql:8.0.21
    ...
+networks:
+  external.group:
+    external: true
```

### Docker Compose B

```diff
services:
  api:
    ...
+   networks:
+     - external.group
+     - default
  db:
    image: mysql:8.0.21
    ...
+networks:
+  external.group:
+    external: true
```

### 既にあるネットワークにJOINする

既にあるネットワークにJOINする場合は下記のような設定をするといける。

```yml
services:
  # ...
networks:
  default:
    name: my-pre-existing-network
    external: true
```

ref. [Networking in Compose \| Docker Documentation](https://docs.docker.com/compose/networking/)
