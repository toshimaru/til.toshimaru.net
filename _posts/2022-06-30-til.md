## Docker mysql で SQLモードを設定

docker compose で設定する場合、下記のような `command` でオプションを指定してやることで、SQLモードを指定可能（sqlモードを無効化する例）。

```yml
services:
  db:
    image: mysql:5.7
    command:
      - --sql_mode=""
```

## キャッシュを使わず docker compose build

`build --no-cache` すればOK．

```console
$ docker compose build --no-cache
```
