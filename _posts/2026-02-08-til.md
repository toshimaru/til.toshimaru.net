## 設定

```console
$ claude mcp add codex codex mcp-server
```

## 確認

claude起動。

```console
$ claude
```

`/mcp` コマンドで codex MCP が認識されていることを確認。

```
❯ /mcp
 Manage MCP servers
 6 servers

   User MCPs (/Users/toshimaru/.claude.json)
 ❯ codex · ✔ connected
```

`✔ connected` と表示されていればOK.
