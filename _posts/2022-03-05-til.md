Rubyの配列操作についてメモ。

## 配列の先頭に要素を追加

```rb
ary = []
ary << 1 #=> [1]
ary << 2 #=> [1, 2]
ary << 3 #=> [1, 2, 3]
```

`push` メソッドでもいける。

```rb
ary = []
ary.push 1 #=> [1]
ary.push 2 #=> [1, 2]
ary.push 3, 4 #=> [1, 2, 3, 4]
```

## 配列の末尾に要素を追加

`unshift`が使える。

```rb
ary = []
ary.unshift 1 #=> [1]
ary.unshift 2 #=> [2, 1]
ary.unshift 3 #=> [3, 2, 1]
ary.unshift 4, 5 #=> [4, 5, 3, 2, 1]
```

## 配列の先頭の要素を削除

`shift`でOK（`unshift`の逆）。

```rb
a = [1,2,3] #=> [1, 2, 3]
a.shift #=> 1
a #=> [2, 3]
```

## 配列の末尾の要素を削除

`pop`でいける。

```rb
a = [1,2,3] #=> [1, 2, 3]
a.pop #=> 3
a #=> [1, 2]
```
