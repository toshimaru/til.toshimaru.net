Rust の Range expressions が便利だった。

```rust
1..5    // 1, 2, 3, 4
1..=5   // 1, 2, 3, 4, 5
..5     // 5 未満
..=5    // 5 以下
1..     // 1 以上
..      // 全範囲
```
