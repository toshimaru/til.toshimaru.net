
```

```

## [MySQL tc.logが初期化出来ず起動出来ない [ERROR] Can’t init tc log](https://analyzegear.co.jp/blog/442)

MySQL on docker を使っていると、下記エラーがでた。

>  [ERROR] Can’t init tc log とエラーが出ており “tc.log” の初期化が出来ていないようでした。

解決方法は下記の通り。

```console
$ rm -f  /var/lib/mysql/tc.log
```
