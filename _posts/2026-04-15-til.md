Skill を `npx skills` コマンドで追加・管理するのが便利だった。

```console
$ npx skills add https://github.com/github/awesome-copilot --skill gh-cli -a codex -g -y
$ npx skills list -g
Global Skills

find-skills ~/.agents/skills/find-skills
  Agents: not linked
gh-cli ~/.agents/skills/gh-cli
  Agents: Claude Code
```
