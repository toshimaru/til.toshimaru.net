## [Ruby ThreadGroup](https://docs.ruby-lang.org/ja/latest/class/ThreadGroup.html)

> スレッドグループを表すクラスです。グループに属する Thread をまとめて操作することができます。

```rb
tg = ThreadGroup.new
t1 = Thread.new { sleep }
t2 = Thread.new { sleep }
puts "t1 is #{t1}" # => t1 is #<Thread:0x50bef60>
puts "t2 is #{t2}" # => t2 is #<Thread:0x50beed0>
tg.add(t1)
```
